# LaTeX Resume – Sonal Satpute

This project contains a **professional one-page resume created using LaTeX**.
The resume is designed to maintain **clean formatting, proper alignment, and minimal spacing**, making it ideal for **Articleship applications in Chartered Accountant firms**.

---

## 📄 Resume Overview

The resume includes the following sections:

* Career Objective
* Education
* Professional Experience
* Technical Skills
* Languages
* Hobbies & Interests
* Declaration

The layout ensures that:

* All **dates and percentages are right aligned**
* All **information fits neatly within one page**
* The resume maintains a **professional and minimal design**

## 🚀 How to Compile the Resume

### Using Overleaf

1. Go to **https://www.overleaf.com**
2. Create a **New Project**
3. Upload `resume.tex`
4. Click **Compile**
5. Download the generated **PDF resume**

### Using Local LaTeX

Run the following command:

```
pdflatex resume.tex
```

This will generate:

```
resume.pdf
```
